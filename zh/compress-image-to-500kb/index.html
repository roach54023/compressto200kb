常见问题解答

500KB有什么用？
500KB 非常适合电子邮件附件、工作申请照片上传和网络图像，您希望获得良好的质量而不需要太大的文件大小。在 500KB 大小时，典型照片 (1200×900px) 保留了出色的视觉质量 — 清晰的细节、准确的颜色，并且没有可见的压缩伪影。
压缩到 500KB 是否会明显降低质量？
对于大多数照片，压缩到 500KB 会产生在视觉上与原始照片没有区别的结果。我们的二分搜索算法可找到仍适合 500KB 以内的最高 JPEG 质量，因此您可以获得最佳结果。只有非常大的原始图像（例如 20MP+ 相机照片）才会显示任何质量差异。
我的图片压缩到 500KB 时会上传到服务器吗？
不会。所有处理完全使用 HTML5 Canvas API 在您的 Web 浏览器内进行。您的图像永远不会发送到任何服务器，永远不会存储在任何地方，也不会被除您之外的任何人看到。

500KB 和 1MB 压缩有什么区别？
500KB 的图像比 1MB 的图像压缩得更彻底，但对于典型的 Web 和电子邮件使用来说，质量差异通常很小。如果您需要通过电子邮件发送照片或上传到招聘门户，500KB 通常就足够了。如果您需要打印图像或在专业环境中使用它，请改用 1MB 预设。

500KB 用于电子邮件和电子邮件专业用途：逐个平台指南

不同的平台以不同的方式处理图像附件。在通过电子邮件发送照片或上传到专业平台之前，您需要了解以下内容。

给客户发电子邮件

Gmail、Outlook 和 Apple Mail 都支持最大 25MB 的附件，因此 500KB 完全在限制范围内。然而，许多企业电子邮件服务器有更严格的策略 - 有些会阻止超过 1MB 或 500KB 的附件。发送前压缩至 500KB 可确保您的电子邮件即使在限制性的公司网络上也能顺利通过。它还意味着更快的交付速度以及为收件人提供更好的移动体验。

LinkedIn 个人资料和封面照片

LinkedIn 建议个人资料照片尺寸在 400×400 像素到 7680×4320 像素之间，最大文件大小为 8MB。不过，LinkedIn 会在上传时重新压缩您的照片，因此发送 500KB 的 JPEG 可以让您更好地控制最终质量，而不是让 LinkedIn 的算法来决定。 800×800 像素的 500KB 照片在桌面版和移动版 LinkedIn 个人资料上看起来都很清晰。

职位申请门户

大多数企业职位申请门户（Workday、Greenhouse、Lever、Taleo）接受最大 2-5MB 的照片，但一些较旧的 HR 系统上传上限为 500KB。压缩至 500KB 可确保与所有系统的兼容性。对于您的简历/履历照片，500KB、400×400 像素是专业标准——质量足够高，打印时看起来清晰，足够小，不会使文档膨胀。

电商产品照片

对于 Shopify、WooCommerce 和 Etsy 等平台，每个产品图片 500KB 是一个不错的目标。它可以让您的商店快速加载（对于移动购物者很重要），同时保持足够的质量，以便客户查看产品详细信息。
